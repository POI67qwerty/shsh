# 二値化アプリ（ネイティブ・シェル）

このリポジトリは、既存のHTML/CSS/JSを各OSのネイティブアプリでローカル表示するためのシェルです。オフライン対応です。

**構成**
- `www`: 既存のHTML/CSS/JS（iCloudからコピー済み）
- `apple`: macOS/iPad/iPhone 向けのSwiftUI + WKWebView
- `windows`: Windows向けのWPF + WebView2

**次のステップ**
1. Mac/iPad/iPhoneは `apple/README.md` を参照してください。
2. Windowsは `windows/README.md` を参照してください。
